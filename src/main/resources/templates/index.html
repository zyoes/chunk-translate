项目背景

随着人工智能技术的快速发展， 跨语言文档处理需求日益增长。


本项目旨在设计并实现一个基于AI的文档翻译平台， 支持超长文档自动分块处理、目录导航、 实时翻译和结果导出。


系统采用 Vue3 + Spring Boot 架构， 并结合大语言模型实现高质量翻译能力。

Project Background

With the rapid development of artificial intelligence, the demand for cross-language document processing is increasing.


This project aims to design and implement an AI-powered document translation platform, supporting automatic chunking of large documents, navigation, real-time translation, and document export.


The system adopts a Vue3 + Spring Boot architecture and integrates large language models to provide high-quality translation services.
